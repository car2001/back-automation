## 📝 Descripción
Este proyecto contiene una suite de pruebas automatizadas para la API de usuarios de **ServeRest** utilizando:
* **Karate DSL**
* **Java**
* **Maven**
* **JUnit 5**

---

## 🎯 Objetivo
Validar las operaciones CRUD del recurso **usuarios**:
* Listar usuarios.
* Registrar un nuevo usuario.
* Buscar un usuario por ID.
* Actualizar un usuario existente.
* Eliminar un usuario.

  ## 🚀 Instalación

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/car2001/back-automation
    cd back-automation
    ```

2.  **Instalar dependencias de Maven:**
    ```bash
    mvn clean install
    ```
